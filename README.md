## Steps to Run Project

In the project directory, you can run:
```
    `git clone https://github.com/rwagatenga/react-challenge.git`
```

```
    `cd react-challenge`
```

```
    `npm install`
```

### Conditions
[] Please better to have react redux chrome tool extention
